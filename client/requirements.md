## Packages
react-webcam | For handling camera access and capturing frames easily
framer-motion | For smooth animations and transitions
lucide-react | For high-contrast icons
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes safely

## Notes
- The app requires camera permissions.
- Voice synthesis uses the native Web Speech API (window.speechSynthesis), no extra package needed.
- High contrast and large typography are prioritized for accessibility.
- API endpoints: POST /api/scans/analyze for image processing.
